A class to store the entries in a row of a database table.

The table in the database would store the predicted values of the character drawn by the user. This class stores the entries in one row of that table. This class inherits from the Table class in flask_table module.
id¶

Col – Unique ID column in the table.

character¶

Col – Column storing the predicted user drawn character.
